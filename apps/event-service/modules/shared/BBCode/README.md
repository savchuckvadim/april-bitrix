# BBCodeRenderer

React компонент для рендеринга BBCode в JSX с использованием Reactstrap и Bootstrap.

## Установка

Компонент уже включен в проект и не требует дополнительной установки.

## Использование

```tsx
import { BBCodeRenderer } from '@/shared/BBCode';

function MyComponent() {
  return (
    <BBCodeRenderer 
      text="[B]Полужирный[/B] и [I]курсивный[/I] текст" 
      className="my-custom-class" 
    />
  );
}
```

## Поддерживаемые теги

### Базовое форматирование
- `[B]текст[/B]` - полужирный шрифт
- `[I]текст[/I]` - курсивный шрифт  
- `[U]текст[/U]` - подчеркнутый шрифт
- `[S]текст[/S]` - зачеркнутый шрифт

### Ссылки
- `[URL=https://example.com]текст[/URL]` - ссылка

### Цитаты
- `>>текст` - цитата (отображается как `｜текст`)

### Смайлики
- `:D`, `:-D` - 😄
- `;)`, `;-)` - 😉
- `:)`, `:-)` - 😊
- `:(`, `:-(` - 😢
- `:P`, `:-P` - 😛
- `;P`, `;-P` - 😜

### Пользователи
- `[USER=123]Имя пользователя[/USER]` - упоминание пользователя (отображается как Badge)

### Иконки
- `[ICON=https://example.com/icon.png title=Описание]` - иконка

### Цвет и размер
- `[COLOR=#ff0000]текст[/COLOR]` - цвет текста
- `[SIZE=16]текст[/SIZE]` - размер текста

## Props

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `text` | `string` | - | BBCode текст для рендеринга |
| `className` | `string` | `''` | Дополнительные CSS классы |

## Особенности

### Автоматическое исправление ошибок
Компонент автоматически исправляет распространенные ошибки в BBCode:

- Неправильные закрывающие теги: `/COLOR]` → `[/COLOR]`
- Теги без открывающей скобки: `COLOR=#ff0000]` → `[COLOR=#ff0000]`
- Одиночные закрывающие скобки

### Улучшенный парсинг атрибутов
- Поддерживает атрибуты без пробелов: `[COLOR=#ff0000]`
- Поддерживает множественные атрибуты: `[ICON=url title=text]`
- Автоматически удаляет лишние кавычки

### Вложенные теги
Поддерживает вложенные BBCode теги любой глубины:

```tsx
<BBCodeRenderer text="[COLOR=#ff0000][B][I]Вложенный текст[/I][/B][/COLOR]" />
```

## Примеры

### Базовое форматирование
```tsx
<BBCodeRenderer text="[B]Полужирный[/B] и [I]курсивный[/I] текст" />
```

### Цветной текст
```tsx
<BBCodeRenderer text="[COLOR=#ff0000]Красный текст[/COLOR] и [COLOR=#00ff00]зеленый текст[/COLOR]" />
```

### Ссылки и пользователи
```tsx
<BBCodeRenderer text="[USER=123]Иван Петров[/USER] написал: Посетите [URL=https://example.com]наш сайт[/URL]" />
```

### Смайлики
```tsx
<BBCodeRenderer text="Привет! :D Как дела? ;-)" />
```

### Сложное форматирование
```tsx
<BBCodeRenderer text="[SIZE=18][B][COLOR=#0000ff]Синий заголовок большого размера[/COLOR][/B][/SIZE]" />
```
