# Посадочная страница April CRM

Полнофункциональная посадочная страница для сервиса April CRM — полная настройка CRM (Битрикс) под ключ.

## Структура

Страница состоит из следующих компонентов:

1. **Header** - Навигация с якорными ссылками, кнопками "Вход в ЛК" и "Зарегистрироваться", sticky при прокрутке
2. **Hero** - Главный баннер с заголовком, подзаголовком и CTA кнопками
3. **Features** - Блок "Что мы делаем" с 4 карточками преимуществ
4. **CommercialProposal** - Блок коммерческого предложения с возможностью скачать PDF или заказать на email
5. **KPIBlock** - Описание отчета KPI с демо-модальным окном
6. **KanbanBlock** - Описание автоматизации сделок и канбана
7. **ListsBlock** - Описание списков (KPI, История, Презентации)
8. **HowItWorks** - Процесс работы в 5 шагов
9. **Testimonials** - Кейсы и отзывы клиентов
10. **Pricing** - Блок с пакетами и ценами
11. **FAQ** - Часто задаваемые вопросы с аккордеоном
12. **LeadForm** - Форма заявки с валидацией и интеграцией в Bitrix
13. **Footer** - Футер с контактами и ссылками

## Технические детали

- **Framework**: Next.js 14+ (App Router)
- **UI Library**: shadcn/ui компоненты из `@workspace/ui`
- **Стилизация**: Tailwind CSS
- **Адаптивность**: Mobile-first подход, responsive на всех устройствах
- **SEO**: Мета-теги, Open Graph, Twitter Card

## Интеграции

### Форма заявки

Форма отправляет данные в `/api/bitrix/lead` (нужно создать endpoint). Payload включает:
- Данные формы (имя, компания, email, телефон, количество менеджеров, комментарий)
- UTM параметры из URL
- Источник (landing_page)
- Timestamp

### Аналитика

Поддерживается трекинг событий для:
- Google Analytics (gtag)
- Яндекс.Метрика (ym)

События:
- `download_cp` - скачивание КП
- `form_submit` - отправка формы заявки

## Настройка

### 1. Добавить зависимость для Accordion

```bash
cd packages/ui
pnpm add @radix-ui/react-accordion
```

### 2. Создать API endpoint для формы

Создать файл `apps/bitrix/app/api/bitrix/lead/route.ts` для обработки заявок и отправки в Bitrix CRM.

### 3. Добавить PDF файл КП

Поместить PDF файл коммерческого предложения в `public/commercial-proposal.pdf` и обновить ссылку в `CommercialProposal.tsx`.

### 4. Настроить контакты

Обновить контактную информацию в `Footer.tsx` (телефон, email, адрес).

### 5. Настроить аналитику

Добавить ID счетчиков в `LeadForm.tsx`:
- Google Analytics: через gtag
- Яндекс.Метрика: заменить `12345678` на реальный ID
